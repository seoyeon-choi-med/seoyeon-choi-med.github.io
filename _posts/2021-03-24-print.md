### 1. 정수의 출력
---
정수를 출력할 때는 %d 서식지정자를 사용합니다.
~~~c
#include <stdio.h> 

int main(void) {
  int a, b;
        printf("a입력: ");
        scanf("%d%d", &a, &b);
        printf("a=%d\n", a);
        printf("b=%d\n", b);
  return 0;
  }
~~~

[출력결과]
a 입력: 3 4
a = 3
b = 4
